# skinseek
minecraft skin download tools

```
 Minecraft
 ╔═╗┬┌─┬┌┐┌  ╔═╗┌─┐┌─┐┬┌─
 ╚═╗├┴┐││││  ╚═╗├┤ ├┤ ├┴┐
 ╚═╝┴ ┴┴┘└┘  ╚═╝└─┘└─┘┴ ┴

Usage of skinseek:
  -n string
        get from player name, example: MaxChen_CX
  -o string
        output filename or path, example: myskin.png; D:\myskin.png
  -u string
        get from player uuid, example: c343a67ff3fd49059e8ff58385a0bbfe
```
